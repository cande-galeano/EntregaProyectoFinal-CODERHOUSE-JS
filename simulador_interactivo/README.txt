BARBER APP
Es una aplicación web donde los usuarios van a poder gestionar sus turnos y/o obtener el presupuesto de los servicios a contratar.
También va a contar con un usuario system, que visualizará los turnos creados por los usuarios y podrá agregar o eliminar turnos.
Para acceder a este usuario y sus funcionalidades, se deberá registrar con el correo system@system.com.
Esta aplicación fue desarrollada con HTML, CSS y JavaScript, utilizando los siguientes conceptos:
- Variables
- Condicionales
- Ciclos/Iteraciones
- Funciones
- Programación orientada a objetos
- Arrays
- Storage y JSON
- DOM
- Eventos
- Jquery
